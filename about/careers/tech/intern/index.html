Internships At Seedlink Shanghai

Internships are open at Seedlink Shanghai in the following areas:

  • Data Science
  • Frontend
  • Backend

Perks of the Jobs

  • Practical experience writing and shipping code
  • Mentoring by experienced colleagues
  • Interesting and highly technical projects
  • Professional colleagues that are dynamic, innovative and highly committed to making a positive impact
  • International startup environment
  • Located near Jing An Temple

Requirements For All Internships

  • Majoring in Data Science, Machine Learning, Computer Science, Statistics, Maths, Linguistics or related fields
  • Good English communication skills
  • Team player
  • Creative thinker
  • Strong problem solving and analytical skills
  • Final year/Post-graduate student
Data Science Interns Backend Interns Frontend Interns

Data Science Interns

Working closely with experienced researchers, you’ll provide expertise as input to computational linguistics projects. You’ll design and implement experiments in Python on Mac/Linux. The goal is to improve our products in the areas of efficiency, accuracy and utility.

Requirements

  • Good Python programming skills: scipy, scikit-learn, numpy, gensim, keras
  • Experience with: machine learning, neural networks, text processing
  • Nice to have: text understanding, classification, pattern recognition, recommendation systems, semantics, syntax, morphology, phonetics, phonology, discourse or similar.
Apply now

Backend Interns

Working with our researchers you’ll develop the brain of our product. This includes implementing algorithms to analyse big data and produce predictive models and designing APIs to provide services to our frontend clients.

Requirements

  • Python/Django skills
  • Experience with Python’s ML libraries: scipy, scikit-learn, numpy, gensim, keras
  • Linux (Ubuntu)
  • Shell scripting
  • REST APIs (Django REST Framework)
  • GraphQL
  • Version control (git)
Apply now

Frontend Interns

You’ll develop our client and consumer facing frontend products with a strong focus on usability and responsiveness. Working in a small team and collaborating with our backend developers, designers and product managers to create a great product.

Requirements

  • Modern frontend skills (React/Redux/Relay, HTML5/CSS3/SASS/Stylus)
  • Webpack, nodejs, npm
  • REST APIs
  • GraphQL
  • Version control (git)
Apply now
